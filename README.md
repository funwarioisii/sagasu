# sagasu

search from any resources(constructed by URI and sentence)

## will support

looking for
  - my likes in Twitter
  - my speaking in Slack
  - my note in Scrapbox


# how to run
 1. `poetry build`
 2. `pip install -U .`
 3. `sagasu`
